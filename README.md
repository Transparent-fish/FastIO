## 🚀 功能特性
- ⚡ 比标准 `cin/cout` 快 3-5 倍的 I/O 性能
- 📦 可配置缓冲区大小（默认 128KB）
- 🖥️ 兼容 `cin/cout` 语法（支持运算符重载）
